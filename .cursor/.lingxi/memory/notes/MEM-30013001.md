# 用户说 hello 时回复 yeah

## Meta

- **Id**: MEM-30013001
- **Kind**: principle
- **Status**: active
- **Strength**: validated
- **Scope**: narrow
- **Audience**: personal
- **Portability**: project-only
- **Source**: manual
- **Tags**: greeting, reply

## When to load

用户说「hello」或类似打招呼时，需要决定如何回复。

## One-liner (for injection)

当用户说 hello 时，回复 yeah！[MEM-30013001]

## Context / Decision

- **Decision**: 用户希望固定回复约定——输入 hello 则输出 yeah！。
- **Signals**: 用户输入为「hello」（或明确等同表述）。
- **Counter-signals**: 非打招呼场景或用户未说 hello 时不适用。
