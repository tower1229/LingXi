## One-liner (for injection)

lingxi-memory 输入契约不包含 context 参数；remember 调用时仅传结构化 input 与（按需）confidence，不提及 context 补充。

## Context / Decision

- **Decision**: 从输入契约中删除 context 参数，避免来源不清导致可验证性下降；同步约束：remember 调用 lingxi-memory 时不再提及 context 补充，仅允许结构化 input 与（按需）confidence。
- **Signals**: 用户明确决策；需求可验证性优先于“按需补充上下文”。
- **Alternatives**: 保留 context 作为可选参数易导致来源模糊、难以验证沉淀依据。
- **Counter-signals**: 无。
